# Billino Desktop Icons

Place your application icons in this directory:

- `icon.ico` – Windows icon (256×256, multi-resolution)
- `icon.icns` – macOS icon
- `icon.png` – Linux / general use (512×512 or 1024×1024)
- `128x128.png` – Small icon variant
- `32x32.png` – Taskbar icon variant

These icons are used by electron-builder for the installer and app window.
